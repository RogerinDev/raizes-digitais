Raízes Digitais: Cultivando a Inovação com IA Generativa

Desenvolvimento de um Assistente Virtual Agrícola para a cafeicultura sul-mineira.

Sobre o Projeto

O projeto visa democratizar o acesso a informações agronômicas através de um chatbot inteligente, estruturado com uma arquitetura RAG (Retrieval-Augmented Generation) para garantir precisão técnica e evitar alucinações da IA.

Tecnologias Utilizadas
